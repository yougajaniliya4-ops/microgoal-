## 🚀 Super Quick Start (5 minutes)

If you're in a hurry:

### 1. Prerequisites
- [ ] Node.js installed
- [ ] MongoDB running (local or Atlas)

### 2. Configure
```bash
# Backend/.env
PORT=8000
MONGO_URL=mongodb://localhost:27017/microgoals
SECRET_KEY=your_secret_here

# Frontend/.env
VITE_API_URL=http://localhost:8000
```

### 3. Install & Run
```bash
# Terminal 1
cd Backend
npm install
npm start

# Terminal 2
cd Frontend
npm install
npm run dev
```

### 4. Open Browser
```
http://localhost:5173
```

**Done!** App is running.
